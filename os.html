Major Operating Systems

Microsoft Windows

Windows is a family of proprietary graphical operating systems developed by Microsoft. The latest version is Windows 11.

Market Share: ~75% of desktop computers

Key Features: GUI-based, extensive software compatibility, DirectX for gaming

macOS

macOS is the operating system developed by Apple Inc. for their Macintosh computers. It's based on Unix.

Market Share: ~15% of desktop computers

Key Features: Sleek UI, tight integration with Apple ecosystem, Unix foundation

Linux

Linux is a family of open-source Unix-like operating systems based on the Linux kernel.

Market Share: ~2% of desktop computers, but dominates servers (~90%)

Key Features: Open-source, highly customizable, secure, many distributions available
